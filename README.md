# Multi-Asset Monte Carlo Options Engine

An institutional-grade, high-performance Monte Carlo pricing and risk management engine built in Python. This project simulates millions of stochastic paths to price complex multi-asset derivatives (e.g., a 50-asset S&P 500 Basket Option) and calculates multi-dimensional risk metrics (Greeks) using real-world market data.

## Key Features

* **High-Performance Computation:** Core stochastic differential equations (SDEs) and matrix algebra are JIT-compiled using `Numba` (`@njit`, `prange`), enabling the simulation of 100+ million discrete asset steps in seconds.
* **Advanced Risk Math (Greeks):** Calculates stable multi-asset Delta vectors using finite difference paired with **Common Random Numbers (CRN)** to eliminate simulation noise.
* **Variance Reduction:** Implements **Antithetic Variates** across multi-dimensional arrays to ensure rapid mathematical convergence.
* **Real-World Calibration:** Integrates with `yfinance` to dynamically ingest historical stock data, handling forward-filling for sparse data and generating positive semi-definite **Cholesky decompositions** for $N \times N$ correlation matrices.
* **SOLID Architecture:** Object-Oriented design decouples the SDE process, random number generation, and instrument payoff logic, making it trivial to add new exotic options (Asian, Barrier, Lookback).
